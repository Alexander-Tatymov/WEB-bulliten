# E-library

Автор: Татымов Александр, 25ИС-11Д

## Описание
Учебный проект: электронная библиотека.

## План функций
- Добавить книгу в фонд
- Показать список выдачи книг из фонда
- Изменить статус конкретной книги 
- Выдача искомой литературы
- Списание устаревшей литературы

## Стек
C# / .NET / Visual Studio
