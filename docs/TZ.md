## 1. Название
bulletin board (Доска объявления)

## 2. Цель
Программа помогает хранить список объявлений и задач и менять их статус.

## 3. Пользователь
Обычный пользователь ПК.

## 4. Функциональные требования
- FR1: Добавить задачу и/или объявлений
- FR2: Показать список задач и/или объявлений 
- FR3: Изменить статус задачи и/или объявлений (Новая/В работе/Готово и Новая/Актуально и/или Неактуально)
- FR4: Удалить задачу и/или объявление

## 5. Нефункциональные требования 
- Приложение запускается без ошибок
- Понятные сообщения об ошибках
- Данные будут храниться локально (позже: JSON-файл)

## 6. Модель данных (черновик)
**Задача Task** **Объявление Announcement**
- Id_task (число)
- Title_task (строка, не пустая)
- Status (New / InProgress / Done)
- Id_announcement (число)
- (New/Revelent and/or Non-revelent)


## 8. План модулей (черновик)
- Core (модели и логика)
- App (консольное меню)
- Storage (позже: сохранение/загрузка)
